# 🎉 بوت فعاليات Discord المتقدم

بوت فعاليات Discord شامل يتيح لك إنشاء وإدارة الفعاليات المختلفة مع نظام نقاط وإحصائيات متقدم.

## ✨ المميزات

### 🎮 الفعاليات
- 🎲 **رولة النرد** - لعبة الحظ البسيطة
- 🎁 **توزيع الجوائز** - Giveaway مع تحديد عدد الفائزين
- 🧠 **أسئلة ثقافية** - مسابقات معرفية
- 🖼️ **خمن الصورة** - ألعاب التخمين البصري
- ⚡ **سباق الكتابة** - تحديات السرعة
- 🎮 **تحديات الأزرار** - ألعاب بالأزرار التفاعلية

### 🎯 النظام المتقدم
- 🏆 **نظام النقاط** - اكسب نقاط من كل فعالية
- 🥇 **لوحة الترتيب** - ترتيب المستخدمين حسب النقاط
- 📊 **الإحصائيات** - تتبع مشاركاتك وإنجازاتك
- 🎖️ **الإنجازات والأوسمة** - حقق إنجازات خاصة
- ⏰ **المؤقتات التلقا��ية** - فعاليات بوقت محدد
- 📝 **لوحة التحكم** - إدارة الفعاليات بسهولة

## 🚀 البدء السريع

### المتطلبات
- Python 3.8+
- pip
- حساب Discord Developer

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/dzsimo275-cloud/.env.git
cd .env

# تثبيت المكتبات
pip install -r requirements.txt

# إعداد متغيرات البيئة
cp .env.example .env
```

### إضافة التوكن

1. افتح [Discord Developer Portal](https://discord.com/developers/applications)
2. اضغط "New Application"
3. اذهب إلى "Bot" واضغط "Add Bot"
4. انسخ التوكن
5. فعّل "Message Content Intent"
6. ألصق التوكن في `.env`

### تشغيل البوت

```bash
python main.py
```

## 📖 أوامر البوت

### 🎲 الفعاليات

#### رولة النرد
```
/dice [عنوان] [وصف]
```
أنشئ فعالية رولة نرد بسيطة

#### توزيع الجوائز
```
/giveaway [الجائزة] [الوصف] [المدة] [عدد الفائزين]
```
أنشئ مسابقة توزيع جوائز

#### أسئلة ثقافية
```
/quiz [العنوان] [السؤال] [الإجابة] [الخيارات]
```
أنشئ سؤال ثقافي

### 🎮 الألعاب

#### خمن الصورة
```
/image_guess [رابط_الصورة] [الإجابة]
```

#### سباق الكتابة
```
/typing_race [النص] [المدة]
```

#### تحدي الأزرار
```
/button_challenge [العنوان] [التعليمات]
```

### 📊 الإحصائيات

#### لوحة الترتيب
```
/leaderboard
```
عرض أفضل 10 لاعبين

#### إحصائياتي
```
/my_stats
```
عرض إحصائياتك الشخصية

#### إحصائيات الفعاليات
```
/events_stats
```
عرض إحصائيات الفعاليات النشطة

### ⚙️ الإدارة

#### إنهاء فعالية
```
/end_event [معرّف_الفعالية]
```

#### إلغاء فعالية
```
/cancel_event [معرّف_الفعالية]
```

#### إعلان
```
/announce [العنوان] [الرسالة]
```

## 📁 هيكل المشروع

```

├── main.py                 # ملف البوت الرئيسي
├── config.py              # الإعدادات
├── models.py              # نماذج البيانات

├── .env.example          # مثال على متغيرات البيئة
├── .gitignore            # ملف التجاهل

    ├── events.py         # كوج الفعاليات

    ├── stats.py          # كوج الإحصائيات
    └── admin.py          # كوج الإدارة
```

## 🎨 التخصيص

### تغيير الألوان
عدّل `config.py`:
```python
COLORS = {
    'primary': 0x5B9BD5,      # اللون الأساسي
    'success': 0x70AD47,      # لون النجاح
    # ...
}
```

### تغيير النقاط
```python
EVENT_CONFIG = {
    'points_per_win': 10,        # نقاط الفوز
    'points_per_participation': 1, # نقاط المشاركة
}
```
