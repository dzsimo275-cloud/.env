# 🎉 بوت فعاليات Discord المتقدم (Node.js)

بوت Discord شامل مكتوب بـ **Node.js** يتيح لك إنشاء وإدارة الفعاليات المختلفة مع نظام نقاط وإحصائيات متقدمة.

## ✨ المميزات

### 🎮 الفعاليات
- **🎲 رولة النرد** - لعبة الحظ البسيطة
- **🎁 توزيع الجوائز** - Giveaway مع حد زمني
- **🧠 أسئلة ثقافية** - مسابقات معرفة
- **🖼️ خمن الصورة** - ألعاب التخمين البصرية
- **⚡ سباق الكتابة** - تحديات السرعة
- **🎮 تحديات الأزرار** - ألعاب تفاعلية

### 🏆 النظام المتقدم
- **🎯 نظام النقاط** - اكسب نقاط من كل فعالية
- **🥇 لوحة الترتيب** - ترتيب المستخدمين حسب النقاط
- **📊 الإحصائيات** - متتبع شامل للمشاركات
- **🏅 الإنجازات والأوسمة** - حقق إنجازات خاصة
- **⏰ المؤقتات التلقائية** - فعاليات بوقت محدد
- **📝 لوحة التحكم** - إدارة الفعاليات بسهولة

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn
- حساب Discord Developer

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/dzsimo275-cloud/.env.git
cd .env

# تثبيت المكتبات
npm install

# إعداد ملف البيئة
cp .env.example .env
```

### إضافة التوكن

1. انتقل إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. اضغط "New Application"
3. اذهب إلى "Bot" واضغط "Add Bot"
4. انسخ التوكن تحت "TOKEN"
5. فعّل "Message Content Intent" و "Server Members Intent"
6. أضف التوكن في ملف `.env`:

```env
DISCORD_TOKEN=your_token_here
```

### تشغيل البوت

```bash
npm start
```

أو للتطوير:
```bash
npm run dev
```

## 📖 أوامر البوت

### 🎮 الفعاليات

#### رولة النرد
```
/dice [عنوان] [وصف]
```
أنشئ لعبة رولة نرد بسيطة

#### توزيع الجوائز
```
/giveaway [الجائزة] [الوصف] [المدة] [الفائزون]
```
أنشئ مسابقة توزيع جوائز

#### أسئلة ثقافية
```
/quiz [العنوان] [السؤال] [الإجابة] [الخيارات]
```
أنشئ سؤال ثقافي

### 📊 الإحصائيات

#### لوحة الترتيب
```
/leaderboard
```
عرض أفضل 10 لاعبين

#### إحصائياتي
```
/my_stats
```
عرض إحصائياتك الشخصية

### ⚙️ الإدارة

#### الإعلانات
```
/announce [العنوان] [الرسالة]
```
أرسل إعلان للخادم (للمسؤولين فقط)

## 🔧 التخصيص

### تغيير الألوان
عدّل `config.js`:
```javascript
export const COLORS = {
  primary: 0x5B9BD5,      // اللون الأساسي
  success: 0x70AD47,      // لون النجاح
  warning: 0xFFC000,      // لون التحذير
  error: 0xC55A11,        // لون الخطأ
};
```

### تغيير النقاط
عدّل `config.js`:
```javascript
export const EVENT_CONFIG = {
  pointsPerWin: 10,        // نقاط الفوز
  pointsPerParticipation: 1, // نقاط المشاركة
};
```
